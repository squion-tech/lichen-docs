## 3 步开始监控

```bash
# 步骤 1：启动 Lichen 服务
docker run -d --name lichen \
  -p 8787:8787 -v lichen-data:/data \
  -e JWT_SECRET=$(openssl rand -hex 32) \
  ghcr.io/squion/lichen:latest

# 步骤 2：打开 Dashboard，注册账号，创建主机并复制 API Token

# 步骤 3：在被监控服务器上安装 Agent
curl -sSL https://lichen.squion.com/install.sh | \
  LICHEN_SERVER_URL=http://your-server:8787 \
  LICHEN_API_TOKEN=tok_xxx \
  LICHEN_HOST_ID=$(uuidgen) \
  bash
```
