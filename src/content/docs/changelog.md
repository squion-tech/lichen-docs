## v0.3.0

**2026-04-18**

### 新功能

- **个人空间与团队空间对齐**：个人空间现支持邀请成员、角色管理、邀请链接，功能与团队空间完全一致
- **邀请链接**（Pro）：生成固定链接，团队成员无需逐个邀请
- **告警聚合**：同一全局规则同时触发多台主机时，合并为一条通知，消除告警轰炸
- **规则 firing 状态**：告警规则列表实时显示「告警中」badge
- **规则备注字段**：支持为告警规则添加上下文说明
- **Webhook 投递日志**：每次通知发送结果（状态码、耗时、错误）可查

### 改进

- Free 套餐主机上限从 3 台调整为 5 台
- 告警历史表格新增「主机」列，主机名可点击跳转
- 告警配置表单底部新增「去配置通知渠道」引导链接

## v0.2.0

**2026-03-15**

### 新功能

- **Agent 离线缓冲**：本地缓冲 ~24h 数据（2880 条），正常关闭持久化到文件，重连后自动补传
- **Docker 容器监控**（Pro）：逐容器 CPU · 内存 · 网络指标，支持历史图表
- **指标数据导出**：CSV · JSON 格式下载
- **主机 7 天可用率**：HostDetail 系统信息卡片显示可用率，颜色区分 ≥99% / ≥95% / <95%
- **邮件验证**：注册后发送验证邮件，Account 页显示未验证横幅
- **密码重置**：忘记密码流程

### 改进

- 自动降采样：>24h 按 5min/桶，>7d 按 1h/桶，含 10000 点安全上限
- 告警迟滞（hysteresis）防抖，HYSTERESIS_PCT = 5%

## v0.1.0

**2026-01-20**

### 首次发布

- Go Agent 单二进制，支持 Linux / macOS，systemd / launchd 服务注册
- 指标采集：CPU · 内存 · 磁盘 · 网络 I/O · 磁盘 I/O · 负载
- 日志监控：tail + 关键词匹配
- 心跳检测：180s 超时判定下线
- 3 种告警类型：指标阈值 / 日志关键词 / 主机离线
- 8 种通知渠道：Telegram · 企业微信 · 飞书 · 钉钉 · Slack · Discord · SMTP · 通用 Webhook
- Webhook 失败重试（3次，指数退避）
- 多租户工作区：邮件邀请 + 角色权限
- GitHub · Google OAuth 登录
- Stripe 计费集成（Free / Pro）
- Docker 自托管：单容器，SQLite
- Cloudflare Workers SaaS：D1 + KV + Queues + Pages
